# CVer-CVF-
提供给CVer用来下载CVF(https://openaccess.thecvf.com/menu)上的论文，不需要在论文列表里面一个一个点开看，有没有与自己相关的论文了！
只需要在code中添加会议首页，关键词，运行之后就可以生成相关信息的excel表格，并且把论文的pdf下载下来
